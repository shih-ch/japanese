1. 基礎句型與指示詞

(1) 肯定、否定與疑問

句型結構例句
肯定A は B です。私わたしは学がく生せいです。(我是學生)
否定A は B じゃありません。学がく生せいじゃありません。(不是學生)
疑問句尾 + か。学がく生せいですか。(是學生嗎？)

(2) 指示代名詞 (こそあど系列)

距離物品 (這/那)場所 (這裡/那裡)連體詞 (+名詞)
近 (我)これ (Kore)ここ (Koko)この (Kono)
中 (你)それ (Sore)そこ (Soko)その (Sono)
遠 (他)あれ (Are)あそこ (Asoko)あの (Ano)
疑 (哪)どれ (Dore)どこ (Doko)どの (Dono)

2. 存在句與位置

地點 + に + 物品/人 + が + あります/います。
  • 部へ屋やに机つくえがあります。(無生命：有書桌)
  • 庭にわに猫ねこがいます。(有生命：有貓)

常用方位：上うえ、下した、中なか、外そと、右みぎ、左ひだり、前まえ、後うしろ、間あいだ、隣となり、近ちかく。

3. 動詞與形容詞變化表 (時態核心)

這是日文學習最重要的文法地圖，請務必熟記。

詞性現在肯定現在否定過去肯定過去否定
動詞食たべます食たべません食たべました食たべませんでした
い形容詞
(熱)		暑あつい暑あつくない暑あつかった暑あつくなかった
な形容詞
(安靜)		静しずかです静しずかじゃありません静しずかでした静しずかじゃありませんでした

4. 請求、願望與穿著

請求與禁止

  • 請做...：動詞(て形) + ください。 (例：食たべてください)
  • 請不要做...：動詞(ない形) + で + ください。 (例：食たべないでください)

願望 (想要...)

  • 想要物品：名詞 + が + ほしいです。 (例：水みずがほしいです)
  • 想要做某事：動詞(去ます) + たいです。 (例：水みずが飲のみたいです)

5. 比較級

A のほうが B より (形容詞) です。

記憶口訣：「のほうが」指著贏的那一方。

  • 猫ねこのほうが犬いぬより好すきです。(比起狗，我更喜歡貓)
